# Parkinson's Disease Detection Using Machine Learning
## Project Overview
 This project focuses on detecting Parkinson's disease using machine learning techniques based on voice measurement data. The aim of the project is to build and compare classification models for predicting whether a person has Parkinson's disease.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Imbalanced-learn

## Machine Learning Models
The following classification algorithms were implemented and compared:

- Decision Tree Classifier
- Support Vector Machine (SVM)

## Project Workflow
1. Data loading and exploration
2. Data preprocessing
3. Feature and target separation
4. Train-test data splitting
5. Feature scaling
6. Class balancing using SMOTE
7. Decision Tree model training
8. SVM model training
9. Model evaluation and comparison

## Model Evaluation
The machine learning models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC

## Key Learning Outcomes
Through this project, I gained practical experience in data preprocessing, handling imbalanced datasets, feature scaling, machine learning model development, model evaluation, and comparing classification algorithms.
